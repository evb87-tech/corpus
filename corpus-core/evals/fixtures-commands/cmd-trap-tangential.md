# User story mapping — introduction rapide

Source : note de veille, avril 2026.

Le user story mapping est une technique de découverte produit proposée par Jeff Patton.
L'idée centrale : organiser les user stories en deux dimensions — horizontalement selon
le flux d'activité de l'utilisateur, verticalement selon la priorité.

Exemple rapide : pour une application de réservation de restaurant, le flux horizontal
pourrait être « chercher → choisir → réserver → payer → évaluer ». Les stories de
haute priorité apparaissent en haut de chaque colonne, les stories moins urgentes plus bas.

Cette technique aide les équipes à visualiser le périmètre d'un produit et à identifier
un MVP cohérent avec le flux utilisateur.
